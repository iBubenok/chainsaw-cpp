# Chainsaw (Rust) → C++ — порт с эквивалентностью поведения 1:1

Этот репозиторий предназначен для портирования инструмента **Chainsaw** с Rust на C++ с сохранением **наблюдаемого поведения 1:1** и обеспечением кроссплатформенности (Windows/Linux/macOS).

## Документы и управление проектом
- Схема идентификаторов, трассируемость и карта Single Source of Truth: `docs/governance/GOV-0001-traceability-and-sot.md`.
- Критерии эквивалентности поведения 1:1 и доказательная база приёмки: `docs/governance/GOV-0002-equivalence-criteria.md`.
- Решения фиксируются в `docs/decisions/` (DEC) и `docs/adr/` (ADR).
- Практическое руководство разработчика (правила разработки/тестов/безопасности): `docs/guide/GUIDE-0001-developer-guide.md`.

## Лицензирование и сторонние компоненты
- Лицензия репозитория порта: см. файл `LICENSE`.
- Реестр лицензий upstream-источников и политика работы со сторонними компонентами: `docs/licensing/`.
- Тексты сторонних лицензий/NOTICE (если требуется): `third_party/`.

## Структура репозитория
- `cpp/` — исходный код C++ порта.
- `tools/` — скрипты и harness для сравнительных прогонов.
- `docs/decisions/` — решения (DEC).
- `docs/adr/` — архитектурные решения (ADR).
- `third_party/` — вендорированные зависимости.
